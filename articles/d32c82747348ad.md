# 1. Javascriptを知る

## Javascriptとは

JavaScriptとは、Webページに様々な動きをつけることができるプログラミング言語です。これまでHTMLやCSSでは実装出来なかった機能がWEBページ上で出来るようになります。

## プログラミング言語とマークアップ言語との違い

HTML/CSS研修で学習したHTMLとCSSはWEBページに掲載する要素(HTML)とレイアウトのデザイン(CSS)を設定する「マークアップ言語」と呼ばれるものでした。これから学ぶプログラミング言語とは、処理方法(機能)をコンピュータに指示する言語です。

## なぜJavascriptなのか

ここまでで、プログラミング言語がWEBページを作成する上で欠かせない存在なのは分かりました。ここからは「なぜJavascriptなのか」という説明をしましょう。

それは、Webブラウザ上で動作するプログラミング言語がJavascriptだけだからです。それはなぜでしょうか？

プログラミング言語で書かれた命令文をコンピュータが理解し実行するためには、その命令文をコンピュータが唯一理解できる機械語に翻訳してくれる翻訳機(一般にエンジンという)が必要です。世の中には数多のプログラミング言語がありますが、Webブラウザに標準で組み込まれているのは、Javascript用のものだけです。そのため、Webページ内の処理はJavascriptで記述するのです。

# 2. Javascriptの基本を学ぶ

## プログラミング言語に触れよう

### ▼ Google Chromeのコンソール画面を使う

この教材では、プログラミングの答え(出力結果)はGoogle Chromeのコンソール画面に表示させます。コンソール画面とはプログラム(処理)の結果を表示してくれる画面のことです。Google Chromeのウィンドウ画面を開いた状態で、F12を押して、実際に立ち上げてみましょう。

画面右下にプログラミングコードを入力するウィンドウがあります。「＞」にカーソルを合わせて記述していきます。
これでプログラミング準備は完了です。それでは文章問題を解いていきましょう。

### ▼ JavaScriptの出力の基本

#### console.log()

`console.log('〇〇');`というコードを書くと、`()`の中の`〇〇`という文字列がコンソールに出力されます。

実際に確かめてみましょう。下記のコードをコンソールに入力し、Enterを押して実行していましょう。

<入力>

```javascript
console.log('Hello, World');
```

<出力>

```bash
Hello, World
```

コードの書き方をより詳しく見ていきましょう。`console.log()`は丸括弧`()`に入力された文字をコンソールに出力します。

また「Hello, World」のような文字のことを**文字列**と呼びます。
文字列は、シングルクォーテーション(')かダブルクォーテーション(")で囲む必要があります。

```javascript
// ダブルクォーテーションでもOK
console.log("Hello, World");

// 囲まないとエラーになる!
console.log(Hello, World);
```

### ▼ 文末のセミコロン

JavaScriptでは、文の終わりにセミコロン（;）を付けます。「処理はこれで終わりです」とコンピュータに伝える為です。
セミコロンがなくても正常に動作する場合が多いですが、思わぬ不具合が生じる可能性があるので忘れずにつけましょう。

### ▼ コメントアウト

文頭に`//`を書くと、その行はコメントとみなされます。そのため、プログラムとしては無視され、実行されません。これをコメントアウトと言います。

## 数値と計算

プログラミングでは、数値を扱うこともできます。数値は文字列と違いクウォーテーションで囲みません。

数値は足し算や引き算といった計算もすることができます。ここでは、代表的な計算である「足し算」「引き算」「掛け算」「割り算」「割り算の余りを求める計算」の5つを紹介します。

数値と記号はすべて半角で記述することに注意しましょう。

```javascript
console.log(3);
// 出力: 3

// 足し算
console.log(5 + 2);
// 出力: 7

// 引き算
console.log(3 - 2);
// 出力: 1

// 掛け算
console.log(3 * 7);
// 出力: 21

// 割り算
console.log(7 / 2);
// 出力: 3.5

// 割り算の余り (7割る3は、2余り1)
console.log(7 % 3);
// 出力: 1
```

`console.log(5 + 2);`は足し算の結果である7が出力されますが、「5 + 2」にクォーテーションをつけると、文字列と解釈されそのまま出力されます。

文字列と数値は明確に違うものであることを意識しましょう。

```javascript
console.log(2 + 5);
// 出力: 7

console.log('2 + 5');
// 出力: 2 + 5

console.log('2' + '5');
// 出力: 25
```

最後の例のように、文字列同士の足し算は文字列の結合を意味します。

## 変数と定数

### なぜ変数を使うのか

実際のプログラムの中には多くの値が登場します。
プログラムを書く時は、それらの値を間違いのないように、そして値の変更に対応しやすいようにすることが求められます。
それを助けるための1つの仕組みが**変数**です。
変数には以下のようなメリットがあります。

1. 同じ値を繰り返し使える
2. 変更に対応しやすい
3. 値の意味がわかりやすい

「1. 同じ値を繰り返し使える」「2. 変更に対応しやすい」という具体例を見てみましょう。
下記のように似た文章を何度も書くのは面倒ですが、変数を使えば同じ値を繰り返し使え、楽にコードを書くことができます。
また文章が後ほど変更されても、変数を用いていれば、代入する値を変更するだけで全ての変更に対応できます。

```javascript
// 「ありがとうございます」の部分を「こんにちは」に変えるなら３箇所全てを変更する必要がある
console.log('Aさん、ありがとうございます');
console.log('Bさん、ありがとうございます');
console.log('Cさん、ありがとうございます');
```

```javascript
// 「ありがとうございます」の部分を「こんにちは」に変えるなら変数に代入する文字列を変更するだけでよい
let message = 'ありがとうございます';
console.log('Aさん、' + message);
console.log('Bさん、' + message);
console.log('Cさん、' + message);
```

### 変数の命名ルール (DRAFT)

変数名の命名にはいくつかのルールがあります。

|変数名|〇△✕|説明|
|:---|:---|:---|
|1number|先頭が数値から始まる変数名はエラーになる|
|番号|△|日本語で書かれた変数。変数にはアルファベットか数字を用いましょう|
|bango|△|ローマ字で書かれた変数。英語で書きましょう|

### 変数は更新できる

変数は、一度代入した値を変更することもできます。
一度値を代入した変数に、その後再び値を代入すると、後に入した値で変数の中身が上書きされます。

定義する時と違って`let`は必要なく、`変数名 = 新しい値`と書けば値が変更されます。

```javascript
let text = 'Hello, World';
console.log(text);
// 出力: Hello, World

text = 'ハロー、ワールド';
console.log(text);
// 出力: ハロー、ワールド
```

この問題を解くためのプログラムを組んでいきましょう。ここで必要な処理は、数値「910」と「1820」の和を求め、文末に文字列「円」を加えて出力する、です。これをプログラミング言語のJavaScriptで表すと、

となります。

## JavaScriptの出力の基礎

### console.log()

```javascript
console.log(910 + 1820 + "円");
```

console.logとは、ブラウザのデバッグコンソールにアクセスし、コードで使用した変数の値を出力するもので、()のなかの値(引数)をconsoleに出力せよという内容のメソッド(関数)※です。

※意味を検索してみましょう！

### console.log()のルール(使い方)

```javascript
console.log(40);
// 出力結果: 40

console.log('おはようございます');
// 出力結果: おはようございます
```

1. 数字のみを出力する場合、括弧の中に数字をそのまま記述します。
2. 文字列を出力する場合、括弧の中の文字をシングルクオーテーション('')またはダブルクオーテーション("")で囲みます。

NOTE: 処理の終わりを伝える


### 演算式

四則演算や割り算の余りを求める式は、JavaScriptでは以下のように表します。

```javascript
// 足し算の解を出力する場合
console.log(2 + 3);
// 出力結果: 5

// 引き算の解を出力する場合
console.log(6 - 3);
// 出力結果: 3

// 掛け算の解を出力する場合
console.log(2 * 3);
// 出力結果: 6

// 割り算の解を出力する場合
console.log(9 / 3);
// 出力結果: 3

// 割り算の余りを出力する場合
console.log(7 % 3);
// 出力結果: 1
```

となります。

これらの演算を表す記号「+, -, *, /, %, ...」を**演算子**、演算を行う式「2 + 3, 6 - 3, ...」を**演算式**と呼びます。

演算子は上記以外にもありますが、後々出てくるので今は割愛します。

試しに以下のコードをそれぞれコンソールに出力してみましょう。

```javascript
console.log('910 + 1820');
// 出力結果: 910 + 1820

console.log(910 + 1820);
// 出力結果: 2730

console.log(910 + 1820 + '円');
// 出力結果: 2730円
```

クオーテーションで囲んでいるため、演算式自体が文字列として出力されます。

計算式の答えを数字で表示したい場合はそのまま数式を入力します。

**数式の結果と文字列は**「+」記号を使うことで繋げて出力することもできます。数式はそのまま記述し、繋げる文字列のみクオーテーションで囲みましょう。
